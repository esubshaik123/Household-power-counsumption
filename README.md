# ⚡ Household Power Consumption Analysis

## 📌 Project Overview
This project analyzes household electricity consumption data using Tableau. It visualizes electricity usage trends and helps understand consumption patterns through interactive charts and dashboards.

## 🚀 Features
- Overall Power Consumption Trend
- Voltage Analysis
- Global Intensity Analysis
- Reactive Power Trend
- Sub Metering Comparison
- Average Daily Consumption
- Interactive Dashboard
- Tableau Story

## 🛠️ Technologies Used
- Tableau Desktop
- CSV Dataset

## 📊 Dashboard Visualizations
- Line Chart
- Bar Chart
- Area Chart
- Trend Analysis
- Comparison Chart

## 📂 Dataset
Household Power Consumption Dataset (CSV)

## 📈 Workflow
1. Collect the dataset
2. Prepare and clean the data
3. Import data into Tableau
4. Create visualizations
5. Design an interactive dashboard
6. Create a Tableau story
